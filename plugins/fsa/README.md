# Full Stack Auth (FSA)

Production-ready authentication for web applications including sign-up, login, logout, and session management.

## Overview

This plugin implements Scalekit Full-Stack Auth (FSA) to handle complete user authentication flows. Once installed, Cursor's agent will help you:

- Set up login, sign-up, and logout flows
- Manage user sessions with secure token handling
- Protect routes and API endpoints
- Implement access control and permissions
- Support multiple frameworks: Next.js, Django, FastAPI, Flask, Go, Spring Boot, Laravel

## Installation

This plugin will be available from the Cursor Marketplace. For local development:

```bash
git clone https://github.com/scalekit-inc/cursor-authstack
```

## Supported Frameworks

- **Node.js**: Next.js, Express
- **Python**: Django, FastAPI, Flask
- **Go**: net/http, Gin
- **Java**: Spring Boot
- **PHP**: Laravel

## Requirements

- Scalekit account ([app.scalekit.com](https://app.scalekit.com))
- Environment variables: `SCALEKIT_ENVIRONMENT_URL`, `SCALEKIT_CLIENT_ID`, `SCALEKIT_CLIENT_SECRET`
